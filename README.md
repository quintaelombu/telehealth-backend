# Backend — Teleconsulta Emilio (Listo para usar)

## Qué hace
- Crea y cobra turnos (Mercado Pago si configuras token; si no, modo demo).
- Genera link de videollamada (Jitsi).
- Panel médico (login).

## Cómo iniciar (Mac/Windows)
- Mac: doble clic `run_mac.command`
- Windows: doble clic `run_windows.bat`

Ambos scripts:
1) Crean entorno, instalan dependencias
2) Ejecutan seed con tus datos (admin = médico)
3) Inician el backend en http://localhost:8000
